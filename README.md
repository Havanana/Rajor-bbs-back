### 社区项目实战
Springboot+Vue前后端分离多用户社区项目

### 代码开源地址
[前端](https://github.com/Havanana/Rajor-bbs-front)
[后端](https://github.com/Havanana/Rajor-bbs-back)

实现了多个用户注册，登录，发帖，回帖，评论，关注，用户中心等功能。
